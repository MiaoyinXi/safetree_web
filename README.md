# 特点
- 支持自动完成手机端登录任务
- 支持自动完成每学期已授课的作业以及专题作业
- 支持自动重置学生密码为123456
- 支持地区专题作业的完成（不在列表内的）
- 支持了 PWA 可以在新版 Chrome 以 app 形式打开
- **`快`**
